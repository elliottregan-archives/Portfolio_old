  • Web Design

    • Expandable Mobile List

      While working at Startfast in Syracuse in 2012, I had the chance to redesign the mobile app, revoPT. At the time, the app used PhoneGap, so I was able to create the front-end entirely in HTML/CSS/JS. When I first came on board, the developers were using a basic jQuery Mobile theme. The app's performance was suffering due to excessive javascript use and other bulk, so I made the decision to scrap jQuery Mobile, and rebuild the front-end from scratch.

      I did away with as much javascript as I could, relying on CSS transitions for all animations to keep the interface smooth, and make the app feel more native.

      Try it out.

    • Knight Scheduler

      Knight Scheduler is a web-app I developed with Herb Susmann.

      Our goal was to help simplify the registration process for students at SUNY Geneseo by providing an easy way to browse the course listings, and automatically creating schedules based on selected courses.

      Building and maintaining Knight Scheduler has been a great experience working with UI design. We have received excellent feedback from students and administrators alike, and we are constantly in the process of refining the user experience.

      www.KnightScheduler.com
